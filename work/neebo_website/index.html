Neebo Website

User Experience Designer

Designed the wireframes for the store locator, store details section, press release section and password reset functionality of the website. The store locator features custom integration with Google maps and enables users to save their preferred location for efficiency in online shopping checkouts. The store details section features current offers and provides customers the ability to view and submit calendar events. The press release section provides users the ability to easily navigate the press release archives. The password reset functionality is focused on ease of use while maintaining strong fraud protection features.
