Farmland Foods Website

User Experience Designer

Designed the wireframes for the home page and recipes section of the website. The user-centered design enables sorting of the recipes by a variety of helpful parameters such as preparation time, cooking method and occasion. Interface design concepts were created for a future phase that will enable the user to save recipes and create a personalized shopping list.
